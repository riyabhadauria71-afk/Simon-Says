# Simon-Says
Description:  
A fun, interactive memory game where players repeat sequences of colors and sounds generated by the computer. Each round adds a new step to the sequence, challenging the player’s memory and attention. The game continues until the player makes a mistake.

Features:

Colorful UI with four interactive buttons (Red, Blue, Green, Yellow).

Randomly generated sequences that increase in difficulty each round.

Sound effects paired with each button for better recall.

Score tracking to measure progress.

Beginner‑friendly code structure for easy learning and customization.

Tech Stack:

Frontend: HTML, CSS, JavaScript

Logic: JavaScript event handling and arrays for sequence storage

Optional Enhancements: Add animations, difficulty levels, or mobile responsiveness

How to Play:

Watch the sequence of colors displayed by the game.

Repeat the sequence by clicking the buttons in the correct order.

Each round adds one more step to the sequence.

Keep playing until you make a mistake — then try again to beat your high score!
